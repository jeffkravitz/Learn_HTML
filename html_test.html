Github commands for terminal
  • cd DIRECTORY_HERE - changes directory
  • cd .. - moves to outer directory
  • ls - lists files/folders in the current directory
  • git clone URL_HERE - clode repository to local computer
  • git add FILE_HERE or, git add -A (adds all) - adds file to github
  • git status - check if changes are committed
  • git commit -m "INSERT_CHANGE_NAME_HERE"
  • git push - pushes active commits
